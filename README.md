# image-convultion-2sem
Структура проекта:
Все изображения хранятся в input/output, в input изначальные изображения,
в output изображения после свертки 
Все изображения взяты с сайта pikwizard.com  
Лицензия:CC0 (Creative Commons Zero)
Основной код проекта хранится в папке src:
    1.convultion.py отвечает функцию за свертки изображения
    2.kernels.py отвечает за ядра свертки
    3.proccessing.py отвечает за обработку краев изображения
В будущем будут добавлены:
    1.CLI интерфейс
    2.Поддержка цветных изображений
    3.Новые ядра свертки
